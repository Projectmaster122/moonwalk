# moonwalk
MoonWalk is a programming language compiled to C++. 
You need g++ to compile MW.
If you don't have g++ in your PATH, the program checks for g++.exe in C:/MinGW/bin.
The provided source is compiled into out.exe.
Example:
```
imp os

noret helloworld()
{
    print("Hello, world!");
}
int test()
{
    return 64;
}
int main() {
    if(yes) {
        print("%d", test());
        exec("regedit");
    }
}
```
